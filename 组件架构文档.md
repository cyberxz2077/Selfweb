# 黄鑫哲个人品牌网站 - 组件架构文档 v1.0

## 一、组件架构概述

### 1.1 组件设计原则

| 原则 | 说明 |
|-----|------|
| **单一职责** | 每个组件只负责一个功能 |
| **可复用性** | 组件可以在不同场景复用 |
| **可组合性** | 组件可以组合成更复杂的组件 |
| **可测试性** | 组件易于单元测试 |
| **可维护性** | 组件代码清晰、易于理解 |

### 2.1 Button 组件

**文件路径**：`components/ui/Button.tsx`

**功能**：统一的按钮组件，支持多种样式

**Props**：

| 属性 | 类型 | 必填 | 默认值 | 说明 |
|-----|------|------|--------|------|
| `variant` | `'primary' \| 'glass' \| 'outline'` | ❌ | `'primary'` | 按钮样式 |
| `size` | `'sm' \| 'md' \| 'lg'` | ❌ | `'md'` | 按钮大小 |
| `isLoading` | `boolean` | ❌ | `false` | 加载状态 |
| `disabled` | `boolean` | ❌ | `false` | 禁用状态 |
| `children` | `ReactNode` | ✅ | - | 按钮内容 |
| `className` | `string` | ❌ | - | 自定义样式 |
| `onClick` | `() => void` | ❌ | - | 点击事件 |
| `href` | `string` | ❌ | - | 链接地址（使用 a 标签） |

**实现**：

```tsx
// components/ui/Button.tsx
'use client';

import React from 'react';
import { cn } from '@/lib/utils';

interface ButtonProps extends React.ButtonHTMLAttributes<HTMLButtonElement> {
  variant?: 'primary' | 'glass' | 'outline';
  size?: 'sm' | 'md' | 'lg';
  isLoading?: boolean;
  href?: string;
}

export function Button({
  variant = 'primary',
  size = 'md',
  isLoading = false,
  disabled,
  className,
  children,
  href,
  ...props
}: ButtonProps) {
  const baseStyles = 'inline-flex items-center justify-center rounded-lg font-medium transition-all duration-300 focus:outline-none focus:ring-2 focus:ring-offset-2 disabled:opacity-50 disabled:cursor-not-allowed';
  
  const variants = {
    primary: 'bg-gradient-to-r from-[#0EA5E9] to-[#0284C7] text-white shadow-lg shadow-[#0EA5E9]/30 hover:shadow-xl hover:shadow-[#0EA5E9]/40 hover:scale-105',
    glass: 'bg-white/10 backdrop-blur-sm border border-white/30 text-[#0EA5E9] hover:bg-white/20 hover:border-[#0EA5E9]/50',
    outline: 'border-2 border-[#0EA5E9] text-[#0EA5E9] hover:bg-[#0EA5E9]/10',
  };
  
  const sizes = {
    sm: 'px-3 py-1.5 text-sm',
    md: 'px-6 py-3 text-base',
    lg: 'px-8 py-4 text-lg',
  };
  
  const classes = cn(baseStyles, variants[variant], sizes[size], className);
  
  if (href) {
    return (
      <a href={href} className={classes} {...(props as any)}>
        {isLoading ? (
          <svg className="animate-spin -ml-1 mr-2 h-4 w-4" xmlns="http://www.w3.org/2000/svg" fill="none" viewBox="0 0 24 24">
            <circle className="opacity-25" cx="12" cy="12" r="10" stroke="currentColor" strokeWidth="4"></circle>
            <path className="opacity-75" fill="currentColor" d="M4 12a8 8 0 018-8V0C5.373 0 0 5.373 0 12h4zm2 5.291A7.962 7.962 0 014 12H0c0 3.042 1.135 5.824 3 7.938l3-2.647z"></path>
          </svg>
        ) : null}
        {children}
      </a>
    );
  }
  
  return (
    <button className={classes} disabled={disabled || isLoading} {...props}>
      {isLoading ? (
        <svg className="animate-spin -ml-1 mr-2 h-4 w-4" xmlns="http://www.w3.org/2000/svg" fill="none" viewBox="0 0 24 24">
          <circle className="opacity-25" cx="12" cy="12" r="10" stroke="currentColor" strokeWidth="4"></circle>
          <path className="opacity-75" fill="currentColor" d="M4 12a8 8 0 018-8V0C5.373 0 0 5.373 0 12h4zm2 5.291A7.962 7.962 0 014 12H0c0 3.042 1.135 5.824 3 7.938l3-2.647z"></path>
        </svg>
      ) : null}
      {children}
    </button>
  );
}
```

**使用示例**：

```tsx
import { Button } from '@/components/ui/Button';

// 主按钮
<Button variant="primary" size="lg">
  查看项目
</Button>

// 玻璃按钮
<Button variant="glass" size="md">
  下载简历
</Button>

// 加载状态
<Button variant="primary" isLoading>
  加载中...
</Button>
```

**文件路径**：`components/ui/GlassCard.tsx`

**功能**：玻璃质感卡片组件

**Props**：

| 属性 | 类型 | 必填 | 默认值 | 说明 |
|-----|------|------|--------|------|
| `children` | `ReactNode` | ✅ | - | 卡片内容 |
| `className` | `string` | ❌ | - | 自定义样式 |
| `hoverEffect` | `boolean` | ❌ | `true` | 悬停效果 |
| `padding` | `'none' \| 'sm' \| 'md' \| 'lg'` | ❌ | `'md'` | 内边距 |

**实现**：

```tsx
// components/ui/GlassCard.tsx
'use client';

import React from 'react';
import { cn } from '@/lib/utils';

interface GlassCardProps {
  children: React.ReactNode;
  className?: string;
  hoverEffect?: boolean;
  padding?: 'none' | 'sm' | 'md' | 'lg';
}

export function GlassCard({
  children,
  className,
  hoverEffect = true,
  padding = 'md',
}: GlassCardProps) {
  const baseStyles = 'bg-white/70 backdrop-blur-2xl border border-white/50 rounded-2xl transition-all duration-300';
  
  const paddings = {
    none: '',
    sm: 'p-4',
    md: 'p-6',
    lg: 'p-8',
  };
  
  const hoverStyles = hoverEffect
    ? 'hover:bg-white/85 hover:border-[#0EA5E9]/30 hover:shadow-[0_20px_25px_-5px_rgba(0,0,0,0.08),0_10px_10px_-5px_rgba(0,0,0,0.02),0_0_0_1px_rgba(14,165,233,0.1)] hover:translate-y-[-4px] hover:scale-[1.01]'
    : '';
  
  const classes = cn(baseStyles, paddings[padding], hoverStyles, className);
  
  return <div className={classes}>{children}</div>;
}
```

**使用示例**：

```tsx

  <h3 className="text-2xl font-bold mb-4">项目标题</h3>
```

### 2.4 Typography 组件

**文件路径**：`components/ui/Typography.tsx`

**功能**：排版组件，统一文字样式

**Props**：

| 属性 | 类型 | 必填 | 默认值 | 说明 |
|-----|------|------|--------|------|
| `variant` | `'h1' \| 'h2' \| 'h3' \| 'p' \| 'small'` | ❌ | `'p'` | 文字类型 |
| `className` | `string` | ❌ | - | 自定义样式 |
| `children` | `ReactNode` | ✅ | - | 文字内容 |

**实现**：

```tsx
// components/ui/Typography.tsx
'use client';

import React from 'react';
import { cn } from '@/lib/utils';

interface TypographyProps {
  variant?: 'h1' | 'h2' | 'h3' | 'p' | 'small';
  className?: string;
  children: React.ReactNode;
}

export function Typography({ variant = 'p', className, children }: TypographyProps) {
  const baseStyles = 'font-sans text-[#1E3A5F]';
  
  const variants = {
    h1: 'text-4xl md:text-6xl font-bold font-display tracking-tight',
    h2: 'text-3xl md:text-4xl font-bold font-display tracking-tight',
    h3: 'text-2xl md:text-3xl font-semibold font-display tracking-tight',
    p: 'text-lg md:text-xl leading-relaxed text-[#1E3A5F]',
    small: 'text-sm leading-relaxed text-[#64748B]',
  };
  
  const classes = cn(baseStyles, variants[variant], className);
  
  switch (variant) {
    case 'h1':
      return <h1 className={classes}>{children}</h1>;
    case 'h2':
      return <h2 className={classes}>{children}</h2>;
    case 'h3':
      return <h3 className={classes}>{children}</h3>;
    case 'small':
      return <small className={classes}>{children}</small>;
    default:
      return <p className={classes}>{children}</p>;
  }
}
```

**使用示例**：

```tsx
import { Typography } from '@/components/ui/Typography';

<Typography variant="h1">黄鑫哲</Typography>
<Typography variant="p">7年产品经理经验，专注AI Agent系统架构</Typography>
<Typography variant="small" className="text-gray-500">
  副标题或补充说明
</Typography>
```

### 3.1 Navbar 组件

**文件路径**：`components/layout/Navbar.tsx`

**功能**：导航栏组件

**Props**：

| 属性 | 类型 | 必填 | 默认值 | 说明 |
|-----|------|------|--------|------|
| `isScrolled` | `boolean` | ❌ | `false` | 是否滚动 |

**实现**：

```tsx
// components/layout/Navbar.tsx
'use client';

import React, { useState, useEffect } from 'react';
import Link from 'next/link';
import { usePathname } from 'next/navigation';
import { Button } from '@/components/ui/Button';

const NAV_LINKS = [
  { name: '首页', href: '/' },
  { name: '项目', href: '/projects' },
  { name: '关于我', href: '/about' },
  { name: '未来展望', href: '/thoughts' },
  { name: '联系方式', href: '/contact' },
];

export function Navbar() {
  const [isScrolled, setIsScrolled] = useState(false);
  const pathname = usePathname();
  
  useEffect(() => {
    const handleScroll = () => {
      setIsScrolled(window.scrollY > 20);
    };
    
    window.addEventListener('scroll', handleScroll);
    return () => window.removeEventListener('scroll', handleScroll);
  }, []);
  
  return (
    <nav
      className={`
        fixed top-0 left-0 right-0 z-50 transition-all duration-300
        ${isScrolled ? 'bg-white/80 backdrop-blur-md shadow-sm py-4' : 'bg-transparent py-6'}
      `}
    >
      <div className="container mx-auto px-6 flex items-center justify-between">
        {/* Logo */}
        <Link href="/" className="font-display text-2xl font-bold tracking-tight text-[#1E3A5F]">
          黄鑫哲
        </Link>
        
        {/* Desktop Menu */}
        <div className="hidden md:flex items-center space-x-8">
          {NAV_LINKS.map((link) => (
            <Link
              key={link.href}
              href={link.href}
              className={`
                font-body text-base transition-colors duration-300
                ${pathname === link.href ? 'text-[#0EA5E9]' : 'text-[#1E3A5F] hover:text-[#0EA5E9]'}
              `}
            >
              {link.name}
            </Link>
          ))}
          
          <Button variant="glass" size="sm" href="/resume.pdf">
            下载简历
          </Button>
        </div>
        
        {/* Mobile Menu Button */}
        <button className="md:hidden text-[#1E3A5F]">
          <svg className="w-6 h-6" fill="none" stroke="currentColor" viewBox="0 0 24 24">
            <path strokeLinecap="round" strokeLinejoin="round" strokeWidth={2} d="M4 6h16M4 12h16M4 18h16" />
          </svg>
        </button>
      </div>
    </nav>
  );
}
```

### 5.2 AnimatedSection 组件

**文件路径**：`components/shared/AnimatedSection.tsx`

**功能**：滚动动画区块组件

**Props**：

| 属性 | 类型 | 必填 | 默认值 | 说明 |
|-----|------|------|--------|------|
| `children` | `ReactNode` | ✅ | - | 区块内容 |
| `className` | `string` | ❌ | - | 自定义样式 |

**实现**：

```tsx
// components/shared/AnimatedSection.tsx
'use client';

import React, { useEffect, useRef, useState } from 'react';
import { cn } from '@/lib/utils';

interface AnimatedSectionProps {
  children: React.ReactNode;
  className?: string;
}

export function AnimatedSection({ children, className }: AnimatedSectionProps) {
  const [isVisible, setIsVisible] = useState(false);
  const ref = useRef<HTMLDivElement>(null);
  
  useEffect(() => {
    const observer = new IntersectionObserver(
      ([entry]) => {
        if (entry.isIntersecting) {
          setIsVisible(true);
          observer.unobserve(entry.target);
        }
      },
      { threshold: 0.1 }
    );
    
    if (ref.current) {
      observer.observe(ref.current);
    }
    
    return () => {
      if (ref.current) {
        observer.unobserve(ref.current);
      }
    };
  }, []);
  
  return (
    <div
      ref={ref}
      className={cn(
        'transition-all duration-700 ease-out',
        isVisible ? 'opacity-100 translate-y-0' : 'opacity-0 translate-y-10',
        className
      )}
    >
      {children}
    </div>
  );
}
```
