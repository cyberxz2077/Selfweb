# 个人作品集网站制作 Prompt

## 一、项目概述

### 1.1 项目背景

为一位拥有 7 年产品经理经验、正在求职 AI 产品经理的候选人搭建个人作品集网站。网站需要展示个人专业能力、项目作品、技能栈，向招聘方传达"懂技术、能落地、有创业者精神"的产品经理形象。

### 1.2 目标用户

- HR 招聘负责人
- 产品总监 / CTO
- AI 创业公司创始人
- 目标城市：杭州 / 上海

### 1.3 核心定位

**"会写代码的产品经理"** —— 既具备传统产品经理的商业洞察和用户体验思维，又具备独立开发 AI 产品的技术能力。

## 二、设计规范

### 2.1 视觉风格

**科技简约几何风**

- 参考网站设计理念：https://www.symbolaudio.com/shop/shop-all
- 核心关键词：简洁、几何感、参数化、非渐变、留白、结构化

### 2.2 配色方案

| 颜色角色 | 色值 | 使用场景 |
|---------|------|---------|
| 主色（深蓝） | `#1E3A5F` | 背景大面积色块、标题、导航 |
| 辅助色（紫） | `#6B4C9A` | 按钮、强调元素、图标 |
| 点缀色（橙） | `#E87B35` | 链接、高亮、交互反馈 |
| 背景色 | `#0A0F1A` | 页面主背景 |
| 卡片背景 | `#141D2B` | 内容卡片、项目展示 |
| 文字主色 | `#F0F4F8` | 正文、标题 |
| 文字次色 | `#8B9AAF` | 辅助说明、注释 |

**重要**：所有颜色均为纯色，拒绝使用渐变。

### 2.3 字体规范

| 字体角色 | 字体选择 | 使用场景 |
|---------|---------|---------|
| 标题字体 | `JetBrains Mono` 或 `Space Mono` | 所有标题、导航、标签 |
| 正文字体 | `Inter` 或 `Noto Sans SC` | 正文内容、说明文字 |
| 强调字体 | `JetBrains Mono` | 数字、代码、技术术语 |

### 2.4 几何与参数化设计

- **几何图形**：使用基础几何形状（矩形、圆形、三角形、多边形）构建视觉元素
- **参数化设计**：通过代码参数控制元素的大小、间距、位置，体现"可编程"的美学
- **网格系统**：采用 12 栏网格系统，内容对齐清晰
- **线条元素**：使用细线条分割内容区域，增强结构感

### 2.5 动画效果

| 动画类型 | 应用场景 | 实现方式 |
|---------|---------|---------|
| 页面进入 | 区块依次淡入 | fade-in + translateY |
| 悬停效果 | 卡片、按钮 | scale(1.02) + 阴影变化 |
| 滚动触发 | 内容区块 | Intersection Observer |
| 微妙动效 | 图标、装饰 | CSS keyframes |

## 五、设计细节参考

### 5.1 参考网站核心特点

**Symbol Audio 设计风格提炼**：

1. **极简主义**：简洁的界面设计，突出内容本身
2. **清晰层级**：主导航 + 内容区块 + 卡片，层级分明
3. **卡片式展示**：产品/项目以卡片形式呈现，带有图片、标题、价格
4. **充足留白**：区块之间保持呼吸感
5. **中性色调**：黑白色为主，辅助特定强调色
6. **网格布局**：规整的内容排列

**迁移到本项目**：

- 采用类似的卡片式项目展示
- 保持清晰的视觉层级
- 使用几何图形作为装饰元素（替代参考网站的家具产品图）
- 深蓝色背景 + 几何装饰 = 独特的"科技感"

### 5.2 几何装饰元素建议

- 矩形色块：作为背景装饰或内容分隔
- 细线条：分割不同内容区域
- 圆形/半圆：作为装饰点缀
- 参数化网格：背景或装饰区域的网格图案
